# The sources of the website of the JuliaCon 2026 Minisymposium Symbolic and Numerical Methods in (Nonlinear) Algebra

## Overview

This is a static website hosted with GitHub pages.

- `assets` -- some logos and images taken from https://github.com/JuliaCon/www.juliacon.org

## Working locally

This website does not require compilation. 

It should be possible to directly edit index.html (and other files) and open it in a browser to observe the changes.
